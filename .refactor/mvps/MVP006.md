# MVP #006: 基础 UI 组件库

## 状态: pending

## 目标
创建可复用的基础 UI 组件。

## 功能边界
- [x] Button 组件（多变体、多尺寸）
- [x] Toast 组件（消息提示）
- [x] ToastManager（Toast 队列管理）
- [x] Dialog 组件（基于 Radix UI）
- [x] Popover 组件（浮动弹窗）
- [x] DropdownMenu 组件

## 不做
- 表单字段组件（单独 MVP）
- 复杂业务组件

## 验收标准
1. 组件能正确渲染
2. 组件支持主题适配
3. Toast 能正确显示/消失

## 依赖
- MVP001（插件骨架）

## 参考代码
```typescript
// src/component/button/Button.tsx
interface ButtonProps {
  variant?: 'default' | 'primary' | 'ghost';
  size?: 'sm' | 'md' | 'icon';
  children: React.ReactNode;
  onClick?: () => void;
  disabled?: boolean;
}

export function Button({ variant = 'default', size = 'md', children, onClick, disabled }: ButtonProps) {
  return (
    <button
      className={`formify-btn formify-btn-${variant} formify-btn-${size}`}
      onClick={onClick}
      disabled={disabled}
    >
      {children}
    </button>
  );
}
```
