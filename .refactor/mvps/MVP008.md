# MVP #008: 条件过滤系统

## 状态: pending

## 目标
实现条件判断和过滤的核心逻辑。

## 功能边界

- [x] FilterType 枚举（group, filter, jsQuery 等）
- [x] OperatorType 枚举（20+ 操作符）
- [x] FilterService 核心匹配逻辑
- [x] 递归条件组合（AND/OR）

2. 能正确评估组合条件（AND/OR）
3. 支持所有基础操作符

## 参考代码
```typescript
// src/model/filter/Filter.ts
export interface Filter {
  id: string;
  type: FilterType;
  operator: OperatorType;
  property?: string;
  value?: any;
  conditions: Filter[];
}

// src/service/filter/FilterService.ts
export class FilterService {
  static match(
    filter: Filter,
    getFieldValue: (property: string) => any
  ): boolean {
    if (filter.type === 'group') {
      const results = filter.conditions.map(c => this.match(c, getFieldValue));
      return filter.operator === OperatorType.And
        ? results.every(r => r)
        : results.some(r => r);
    }
    return this.evaluateOperator(filter, getFieldValue);
  }
}
```
