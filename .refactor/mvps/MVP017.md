# MVP #017: 命令系统

## 状态: pending

## 目标
注册 Obsidian 命令，实现快速打开和创建表单。

## 功能边界
- [x] ApplicationCommandService 类
- [x] "Open Form" 命令（打开表单选择器）
- [x] "Create Form" 命令（创建新表单）
- [x] CFormSuggestModal（表单选择模态框）
- [x] CreateFileModal（创建文件模态框）

## 不做
- 动态命令注册
- 右键菜单

## 验收标准
1. Ctrl+P 能搜索到命令
2. Open Form 能打开选择器
3. 选择表单后能打开

## 依赖
- MVP004（ServiceContainer）
- MVP014（FormService）

## 参考代码
```typescript
// src/service/command/ApplicationCommandService.ts
export class ApplicationCommandService {
  initialize(plugin: FormPlugin): void {
    plugin.addCommand({
      id: 'open-form',
      name: 'Open Form',
      callback: () => {
        new CFormSuggestModal(plugin.app, (file) => {
          plugin.app.workspace.openLinkText(file.path, '', false);
        }).open();
      }
    });

    plugin.addCommand({
      id: 'create-form',
      name: 'Create Form',
      callback: () => {
        new CreateFileModal(plugin.app, plugin.settings.formFolder).open();
      }
    });
  }
}
```
