# MVP #010: 表单字段组件

## 状态: pending

## 目标
实现所有表单字段类型的 UI 组件。

## 功能边界
- [x] CpsFormFieldControl 统一字段渲染组件
- [x] Text 字段组件
- [x] Textarea 字段组件
- [x] Number 字段组件
- [x] Select 字段组件（单选/多选）
- [x] Radio 字段组件
- [x] Checkbox/Toggle 字段组件
- [x] Date/DateTime/Time 字段组件

## 不做
- FileList 字段（单独 MVP）
- Database 字段（单独 MVP）
- PropertyValue 字段（单独 MVP）

## 验收标准
1. 所有基础字段能正确渲染
2. 字段值变更能正确回调
3. 字段支持 placeholder 和 description

## 依赖
- MVP005（数据模型）
- MVP006（UI 组件库）

## 参考代码
```typescript
// src/view/shared/control/CpsFormFieldControl.tsx
export function CpsFormFieldControl({ field, value, onChange }: Props) {
  switch (field.type) {
    case FormFieldType.TEXT:
      return <TextControl field={field} value={value} onChange={onChange} />;
    case FormFieldType.SELECT:
      return <SelectControl field={field} value={value} onChange={onChange} />;
    // ... 其他类型
  }
}
```
