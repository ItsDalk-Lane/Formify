# MVP #015: 表单文件视图

## 状态: pending

## 目标
实现 .cform 文件的渲染视图，支持表单展示和提交。

## 功能边界
- [x] CpsFormFileView（文件视图注册）
- [x] CpsFormRenderView（表单渲染组件）
- [x] CpsFormDataView（数据提供组件）
- [x] 表单提交按钮
- [x] 字段值收集

## 不做
- 表单编辑功能（单独 MVP）
- 预览/编辑切换

## 验收标准
1. .cform 文件能用自定义视图打开
2. 表单字段能正确渲染
3. 点击提交能执行 FormService

## 依赖
- MVP010（字段组件）
- MVP014（FormService）

## 参考代码
```typescript
// src/view/preview/CpsFormRenderView.tsx
export function CpsFormRenderView({ config, onSubmit }: Props) {
  const [values, setValues] = useState<Record<string, any>>({});

  const handleSubmit = async () => {
    await onSubmit(values);
  };

  return (
    <div className="formify-form">
      {config.fields.map(field => (
        <CpsFormFieldControl
          key={field.id}
          field={field}
          value={values[field.id]}
          onChange={(v) => setValues(prev => ({ ...prev, [field.id]: v }))}
        />
      ))}
      <Button variant="primary" onClick={handleSubmit}>
        Submit
      </Button>
    </div>
  );
}
```
