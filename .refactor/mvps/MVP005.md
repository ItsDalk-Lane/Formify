# MVP #005: 表单数据模型

## 状态: pending

## 目标
定义表单系统的核心数据结构和枚举。

## 功能边界
- [x] FormConfig 主接口
- [x] IFormField 基础字段接口
- [x] IFormAction 基础动作接口
- [x] FormFieldType 枚举（15+ 类型）
- [x] FormActionType 枚举（15+ 类型）
- [x] 基础字段类型：ITextField, ITextAreaField, ISelectField 等

## 不做
- Filter 系统（单独 MVP）
- StartupCondition（单独 MVP）
- 复杂字段实现

## 验收标准
1. 类型定义完整无 TypeScript 错误
2. 能正确描述表单结构

## 依赖
- MVP001（插件骨架）

## 参考代码
```typescript
// src/model/FormConfig.ts
export interface FormConfig {
  id: string;
  name: string;
  description?: string;
  fields: IFormField[];
  actions: IFormAction[];
  settings?: FormSettings;
}

// src/model/enums/FormFieldType.ts
export enum FormFieldType {
  TEXT = 'text',
  TEXTAREA = 'textarea',
  PASSWORD = 'password',
  NUMBER = 'number',
  DATE = 'date',
  DATETIME = 'datetime',
  TIME = 'time',
  CHECKBOX = 'checkbox',
  TOGGLE = 'toggle',
  RADIO = 'radio',
  SELECT = 'select',
  PROPERTY_VALUE_SUGGESTION = 'property_value_suggestion',
  FILE_LIST = 'file_list',
  FOLDER_PATH = 'folder_path',
  DATABASE = 'database'
}
```
