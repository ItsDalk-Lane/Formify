# MVP #016: 表单编辑器基础

## 状态: pending

## 目标
实现表单配置的可视化编辑器基础框架。

## 功能边界
- [x] CpsFormEditView 主编辑器组件
- [x] 字段列表编辑（添加、删除、排序）
- [x] 单个字段属性编辑
- [x] 基础字段类型编辑器（Text, Select, Checkbox）
- [x] 防抖保存

## 不做
- 动作编辑器（单独 MVP）
- 复杂字段编辑器
- 导入/导出

## 验收标准
1. 能添加和删除字段
2. 能编辑字段属性
3. 修改后自动保存

## 依赖
- MVP005（数据模型）
- MVP006（UI 组件）
- MVP010（字段组件）

## 参考代码
```typescript
// src/view/edit/CpsFormEditView.tsx
export function CpsFormEditView({ config, onSave }: Props) {
  const [fields, setFields] = useState(config.fields);
  const debouncedSave = useDebounce(fields, 500);

  useEffect(() => {
    onSave({ ...config, fields: debouncedSave });
  }, [debouncedSave]);

  const addField = (type: FormFieldType) => {
    const newField = createDefaultField(type);
    setFields([...fields, newField]);
  };

  return (
    <div className="formify-editor">
      <FieldList fields={fields} onUpdate={setFields} />
      <AddFieldButton onAdd={addField} />
    </div>
  );
}
```
