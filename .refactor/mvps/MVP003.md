# MVP #003: 国际化系统

## 状态: pending

## 目标
实现多语言支持框架，支持英文、简体中文、繁体中文。

## 功能边界
- [x] 定义 Local 接口（所有文本键）
- [x] 实现 en.ts（英文）
- [x] 实现 zh.ts（简体中文）
- [x] 实现 zhTw.ts（繁体中文）
- [x] 创建 localInstance 单例
- [x] 根据 Obsidian 语言设置自动选择

## 不做
- 动态语言切换
- 完整的文本翻译（只添加基础文本）

## 验收标准
1. 不同语言环境下显示对应语言
2. 添加新文本只需修改 Local 接口和实现

## 依赖
- MVP001（插件骨架）

## 参考代码
```typescript
// src/i18n/local.ts
export interface Local {
  // Settings
  settings_title: string;
  settings_form_folder: string;
  settings_script_folder: string;
  // ... 更多文本
}

export const localInstance: Local = getLocalInstance();

function getLocalInstance(): Local {
  const lang = window.localStorage.getItem('language') || 'en';
  switch (lang) {
    case 'zh': return new ZhLocal();
    case 'zh-TW': return new ZhTwLocal();
    default: return new EnLocal();
  }
}
```
