# MVP #007: React Context 系统

## 状态: pending

## 目标
创建 React Context 提供全局状态访问。

## 功能边界
- [x] ObsidianAppContext（提供 App 实例）
- [x] PluginSettingsContext（提供设置）
- [x] ServiceContainerContext（提供服务容器）
- [x] LoopContext（循环状态，用于 Action）

## 不做
- 复杂状态管理
- 状态持久化

## 验收标准
1. Context 能正确传递值
2. 子组件能通过 useContext 获取值

## 依赖
- MVP002（Settings 框架）
- MVP004（ServiceContainer）

## 参考代码
```typescript
// src/context/obsidianAppContext.tsx
import { createContext, useContext } from 'react';
import { App } from 'obsidian';

const ObsidianAppContext = createContext<App | null>(null);

export function ObsidianAppProvider({ app, children }: { app: App; children: React.ReactNode }) {
  return (
    <ObsidianAppContext.Provider value={app}>
      {children}
    </ObsidianAppContext.Provider>
  );
}

export function useApp(): App {
  const app = useContext(ObsidianAppContext);
  if (!app) throw new Error('useApp must be used within ObsidianAppProvider');
  return app;
}
```
