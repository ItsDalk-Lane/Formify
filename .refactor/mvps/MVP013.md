# MVP #013: 基础 Action 实现

## 状态: pending

## 目标
实现最常用的三个动作：CreateFile、InsertText、UpdateFrontmatter。

## 功能边界
- [x] CreateFileActionService（创建文件）
- [x] InsertTextActionService（插入文本）
- [x] UpdateFrontmatterActionService（更新 frontmatter）
- [x] 对应的 Action Model

## 不做
- 其他动作类型
- 动作编辑 UI

## 验收标准
1. CreateFile 能创建带模板的文件
2. InsertText 能在指定位置插入文本
3. UpdateFrontmatter 能正确更新属性

## 依赖
- MVP011（Action Chain）
- MVP012（文件操作）

## 参考代码
```typescript
// src/service/action/create-file/CreateFileActionService.ts
export class CreateFileActionService implements IActionService {
  accept(action: IFormAction): boolean {
    return action.type === FormActionType.CREATE_FILE;
  }

  async run(action: CreateFileFormAction, context: ActionContext, chain: ActionChain): Promise<void> {
    const { filePath, template, openAfterCreate } = action;
    const processedPath = FormTemplateProcessEngine.process(filePath, context.state, context.app);
    const processedContent = FormTemplateProcessEngine.process(template, context.state, context.app);

    await context.services.fileOperationService.writeFile({
      filePath: processedPath,
      content: processedContent,
      openAfterCreate
    });

    await chain.next(context);
  }
}
```
